# crs-json
> JSON of Coordinate Reference System Information

## download link
https://github.com/DanielJDufour/crs-json/raw/main/crs.json.zip

## background
Built using data from [crs-csv](https://github.com/danieljdufour/crs-csv), which uses [epsg.io](https://epsg.io/).
